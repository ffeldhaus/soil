Als Spielleiter registrieren

Um sich als Spielleiter zu registrieren, füllen Sie bitte die folgenden Felder aus. Das Planspiel wird nur Lehrpersonen und Gruppenleitern bereitgestellt. Es ist deshalb erforderlich, dass Sie die Einrichtung, für die Sie tätig sind, angeben. Bitte verwenden Sie, wenn möglich, auch eine E-Mail-Adresse Ihrer Einrichtung.

Der Vorname darf maximal 128 Zeichen enthalten. Der Nachname darf maximal 128 Zeichen enthalten. Die Einrichtung muss mindestens 3 Zeichen enthalten. Bitte verwenden Sie keine Acronyme (z.B. RWTH). Die Einrichtung darf maximal 128 Zeichen enthalten. Bitte geben Sie eine gültige E-Mail Addresse ein. Die E-Mail Addresse darf maximal 128 Zeichen enthalten. Bitte geben Sie Ihr Passwort aus mindestens 8 Zeichen ein. Das Passwort muss mindenstens 8 Zeichen enthalten. Das Passwort darf maximal 128 Zeichen enthalten. Die Passwörter stimmen nicht überein.
